
# Open Source

We actively support noval open source projects requiring HTTP-based hosting services.

# Requirements

When you pick the open source plan, you will be prompted for few additional questions.

- The project title and description must be well described.
- Must provide an active source control URL, e.g.: github, gitlab, etc.
- Must be active on a monthly basis.

# What's included

Everything included in regular plans are part of the open source plan. On demand
services are not available.
